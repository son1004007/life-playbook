# 중장기 목표 운영 문서

이 문서는 생활 루틴 위에 자격증, 이직, 건강, 가정, 커리어 목표를 어떻게 배치할지 정리하는 기준 문서입니다.

핵심은 목표를 많이 세우는 것이 아니라, 생활이 무너지지 않는 범위에서 꾸준히 진행 가능한 구조를 만드는 것입니다.

---

## 1. 목표 운영의 핵심

```text
생활 루틴이 먼저다.
그 위에 자격증, 이직, 운동, 장기 목표를 올린다.
```

중장기 목표는 의지만으로 달성하지 않습니다. 다음 구조가 필요합니다.

```text
목표
→ 실행 장소
→ 반복 행동
→ 점검 주기
→ 수정 기준
```

목표를 관리할 때는 다음 순서를 따릅니다.

```text
1. 생활 유지
2. 건강 회복과 체력 유지
3. 자격증 학습
4. 이직 준비
5. 장기 커리어 확장
```

---

## 2. 현재 목표 분류

| 영역 | 목표 | 운영 방식 | 관련 저장소 |
|---|---|---|---|
| 생활 | 출근 전/퇴근 후 루틴 안정화 | 장소 기반 루틴 | `life-playbook` |
| 건강 | 본사 점심 헬스장 루틴 유지 | 본사 점심 장소 고정 | `life-playbook` |
| 자격증 | PMP 등 자격증 학습 지속 | 지하철 문제풀이, 집 책상 오답정리 | `pmp-study` |
| 이직 | 플랫폼 백엔드 중심 이직 준비 | 공고 탐색, 문서 수정, 면접 준비 | `career-playbook` |
| 업무기록 | 경력기술 후보 축적 | 업무일지와 산출물 기록 | `work-log` |

## 3. 3개월 단위 운영 기준

중장기 목표는 3개월 단위로 점검합니다.

```text
1개월 = 루틴 적응
2개월 = 실행량 확보
3개월 = 결과 점검과 방향 수정
```

### 점검 질문

```text
생활 루틴이 유지되고 있는가?
지하철 문제풀이가 누적되고 있는가?
집 책상에서 오답정리나 이직 문서 작업을 하고 있는가?
본사 점심 운동 루틴이 유지되고 있는가?
이직 준비가 실제 산출물로 쌓이고 있는가?
지금 목표가 현재 상황에 맞는가?
```

## 4. 생활 목표

생활 목표의 핵심은 거창한 하루를 만드는 것이 아니라, 기본이 무너지지 않게 하는 것입니다.

### 최소 기준

```text
아침에 씻고 출근한다.
퇴근 후 손 씻고 옷 갈아입는다.
식사한다.
씻는다.
너무 늦게 자지 않도록 준비한다.
```

### 성공 기준

```text
아침 출근 루틴이 자동화된다.
퇴근 후 침대에 바로 눕는 패턴이 줄어든다.
집 책상에 앉는 횟수가 늘어난다.
```

---

## 5. 건강 목표

건강 목표는 무리한 운동 계획보다 장소 기반 운동 루틴으로 관리합니다.

```text
본사 점심 = 헬스장
다른 근무지 점심 = 식사 + 산책 + 회복
집 = 과한 운동보다 생활 정리와 수면 준비
```

### 운동 최소 기준

```text
본사 점심에 헬스장에 간다.
운동량은 컨디션에 따라 조절한다.
피곤한 날은 샤워와 스트레칭만 해도 성공으로 본다.
```

### 주의 기준

```text
운동을 못한 날을 실패로 보지 않는다.
본사가 아닌 장소에서 헬스장 루틴을 억지로 적용하지 않는다.
수면과 식사가 무너지면 운동보다 회복을 우선한다.
```

## 6. 자격증 목표

자격증 학습은 `pmp-study` 저장소에서 세부 관리합니다.

이 저장소에서는 자격증 공부를 생활 속 어디에 배치할지만 정의합니다.

```text
지하철 = PMI Study Hall 문제풀이
집 책상 = 오답정리 / 개념 보완
카페·도서관 = 긴 공부 / 모의고사
```

### 최소 기준

```text
지하철에서 Practice Question 3개
또는
틀린 문제 1개 해설 읽기
```

### 보통 기준

```text
지하철 문제풀이
+ 집 책상 오답정리 10~25분
```

### 좋은 날 기준

```text
Mini Test 1개
+ 오답정리
+ 약점 개념 보완
```

## 7. 이직 목표

이직 전략과 문서 산출물은 `career-playbook` 저장소에서 관리합니다.

이 저장소에서는 이직 준비를 생활 루틴에 배치하는 기준만 관리합니다.

```text
지하철 = 채용공고 탐색 / 저장
집 책상 = 이력서 한 문단 수정 / 경력기술서 보완
카페·도서관 = 지원서 작성 / 면접 준비
```

### 최소 기준

```text
채용공고 1개 저장
또는
이력서 한 문장 수정
또는
경력기술서 한 줄 보완
```

### 보통 기준

```text
공고 3개 탐색
+ 괜찮은 공고 저장
+ 집 책상에서 문서 25분 수정
```

### 좋은 날 기준

```text
지원할 회사 분석
+ 이력서/경력기술서 수정
+ 실제 지원 또는 면접 답변 정리
```

## 8. 목표 간 우선순위

목표가 충돌하면 다음 순서로 판단합니다.

```text
1. 출근과 가정 생활을 무너뜨리지 않는가?
2. 건강과 수면을 심하게 깎지 않는가?
3. 현재 가장 가까운 마감이 무엇인가?
4. 이직에 직접 연결되는가?
5. 장기적으로 경력 무기가 되는가?
```

### 우선순위 예시

| 상황 | 우선순위 |
|---|---|
| 잠이 부족하고 몸이 무거움 | 회복 > 공부/이직 |
| PMP 시험이 가까움 | 자격증 > 이직 문서 |
| 지원하고 싶은 공고가 있음 | 이직 문서 > 추가 공부 |
| 본사 점심 시간 확보 | 헬스장 > 휴대폰 사용 |
| 가족 일정 있음 | 가정 일정 > 개인 프로젝트 |

## 9. 월간 점검 기준

매월 한 번 다음 내용을 확인합니다.

```md
## 월간 점검

### 생활
- 아침 루틴은 유지됐는가?
- 퇴근 후 루틴은 유지됐는가?
- 침대에서 휴대폰으로 무너지는 시간이 줄었는가?

### 건강
- 본사 점심 헬스장 루틴은 몇 번 실행했는가?
- 수면과 식사는 유지됐는가?

### 자격증
- 문제풀이와 오답정리가 누적됐는가?
- 약점 영역이 줄었는가?

### 이직
- 저장한 공고가 있는가?
- 이력서/경력기술서 수정이 있었는가?
- 실제 지원 또는 면접 준비 산출물이 있는가?

### 다음 달 조정
- 줄일 것:
- 유지할 것:
- 추가할 것:
```

## 10. 운영 원칙

1. 목표보다 생활 구조를 먼저 만든다.
2. 목표는 장소와 연결해야 실행된다.
3. 자격증과 이직 준비를 모두 매일 하려 하지 않는다.
4. 회복도 계획의 일부로 본다.
5. 본사, 지하철, 집 책상, 카페의 역할을 섞지 않는다.
6. 3개월 단위로 목표를 조정한다.
7. 실행량보다 끊기지 않는 구조를 우선한다.
