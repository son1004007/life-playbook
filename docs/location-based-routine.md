# 장소 기반 생활·학습·이직 루틴

이 문서는 시간표나 요일이 아니라 **장소와 행동을 연결하는 방식**으로 생활 루틴을 설계하기 위한 기준 문서입니다.

---

## 1. 핵심 원칙

루틴은 의지로 유지하는 것이 아니라, 반복되는 환경에 행동을 묶어서 유지합니다.

```text
장소가 행동을 결정하게 만든다.
시간표가 아니라 장소표를 만든다.
요일이 아니라 반복 횟수로 관리한다.
```

장소 기반 루틴의 장점은 다음과 같습니다.

```text
판단을 줄인다.
실행 단서를 명확히 한다.
장소가 바뀌면 행동도 자연스럽게 바뀐다.
생활 필수 루틴과 프로젝트 루틴을 분리할 수 있다.
```

---

## 2. 최종 구조

```text
아침:
집 욕실/현관 루틴
→ 씻기
→ 옷 입기
→ 필요한 것 챙기기
→ 출근

이동:
지하철 루틴
→ PMI Study Hall 문제풀이
→ Practice Question / Mini Test
→ 틀린 문제 표시

근무 중:
본사 점심이면
→ 헬스장

다른 근무지 점심이면
→ 식사
→ 산책
→ 회복

퇴근 후:
집 현관/욕실/주방 루틴
→ 손 씻기
→ 옷 갈아입기
→ 식사
→ 씻기

밤:
집 책상 루틴
→ 자격증 오답정리
또는
→ 이직 준비 25분

주말/여유:
카페·도서관 루틴
→ 긴 학습
또는
→ 이직 문서 작업
```

## 3. 장소별 행동표

| 장소 | 역할 | 기본 행동 | 부가 행동 | 금지/주의 |
|---|---|---|---|---|
| 집 현관 | 귀가 전환 | 손 씻기, 옷 갈아입기 | 가방 정리 | 바로 침대 눕기 |
| 집 주방 | 식사 | 식사 | 설거지 조금 | 폰 보며 식사 길게 끌기 |
| 집 욕실 | 회복/정리 | 씻기, 양치 | 빨래 넣기 | 씻기 미루기 |
| 집 책상 | 깊은 처리 | 자격증 오답정리, 이직 준비 | 25분 집중 작업 | 유튜브, 쇼츠 |
| 침대 | 수면 | 잠 | 짧은 독서 | 공부, 이직 준비, 문제풀이 |
| 지하철 | 가벼운 실행 | PMI Study Hall 문제풀이 | 오답 표시, 공고 저장 | 깊은 공부, 이력서 작성 |
| 본사 점심 | 운동 | 헬스장 | 샤워, 식사 조정 | 점심시간을 폰으로 소진 |
| 다른 근무지 점심 | 회복 | 식사, 산책 | 짧은 복습 | 헬스장 압박 |
| 카페·도서관 | 긴 작업 | 긴 학습, 이직 문서 작업 | 모의고사, 면접 준비 | 의미 없는 인터넷 탐색 |

## 4. 집 루틴

집은 하나의 장소가 아니라 여러 구역으로 나누어 관리합니다.

```text
현관 = 전환
욕실 = 씻기
주방 = 식사
책상 = 처리
침대 = 수면
```

### 출근 전

```text
기상
→ 씻기
→ 옷 입기
→ 필요한 것 챙기기
→ 출근
```

아침에는 공부나 이직 준비를 많이 넣지 않습니다. 아침의 목적은 출근을 안정적으로 하는 것입니다.

가능하면 다음 정도만 추가합니다.

```text
물 마시기
전날 틀린 문제 1개 보기
```

### 퇴근 후

```text
집 도착
→ 손 씻기
→ 옷 갈아입기
→ 식사
→ 씻기
→ 책상 또는 휴식
```

퇴근 후에는 바로 생산적이려고 하지 않습니다. 먼저 몸을 집 모드로 전환하고, 식사와 씻기를 처리합니다.

### 집 책상 시작 루틴

```text
책상에 앉기
→ 타이머 10분 또는 25분
→ 오늘 할 작업 하나만 열기
```

자격증이면:

```text
PMI Study Hall 오답 1개 보기
또는
개념 하나 정리
```

이직 준비면:

```text
이력서 파일 열기
또는
공고 하나 저장
또는
경력기술서 한 줄 수정
```

## 5. 지하철 루틴

지하철은 깊은 공부를 하는 곳이 아니라, 가벼운 문제풀이와 탐색을 하는 곳입니다.

```text
자리에 앉음
→ PMI Study Hall 열기
→ Practice Question 또는 Mini Test
→ 틀린 문제 표시
```

상태별 기준은 다음과 같습니다.

| 상태 | 행동 |
|---|---|
| 앉아 있음 / 15분 이상 안정적 | Mini Test |
| 서 있음 / 사람 많음 | Practice Question 몇 개 |
| 환승 많음 | 오답 해설 읽기 |
| 너무 피곤함 | 쉬기 |
| 집중 잘 됨 | 틀린 문제 북마크 |

성공 기준은 낮게 둡니다.

```text
Practice Question 3개 풀면 성공
Mini Test 1개 풀면 좋음
틀린 문제 1개 해설 읽으면 최소 성공
```

## 6. 본사 점심 루틴

본사 사무실에서만 헬스장에 갈 수 있다면, 본사 점심은 운동 루틴으로 고정합니다.

```text
본사 출근
→ 점심시간
→ 헬스장
```

운동량은 컨디션에 따라 조절합니다.

| 상태 | 기준 |
|---|---|
| 좋음 | 근력운동 + 짧은 유산소 |
| 보통 | 주요 운동 2~3개 |
| 피곤함 | 걷기, 스트레칭, 샤워만 |
| 시간이 부족함 | 헬스장에 들어갔다 나오기만 해도 성공 |

핵심은 다음입니다.

```text
본사 점심에는 일단 헬스장에 간다.
운동량은 그날 조절한다.
```

---

## 7. 다른 근무지 점심 루틴

다른 근무지에서는 헬스장 루틴을 억지로 적용하지 않습니다.

```text
식사
→ 10분 걷기
→ 커피 또는 휴식
→ 오후 업무 복귀
```

다른 근무지는 운동 장소가 아니라 회복 장소로 봅니다.

```text
본사 = 운동 장소
다른 근무지 = 회복 장소
```

---

## 8. 자격증 학습 루프

자격증 학습은 장소별로 역할을 나눕니다.

```text
지하철 = 문제풀이
집 책상 = 오답정리 / 개념 보완
카페·도서관 = 긴 공부 / 모의고사
```

학습 흐름은 다음과 같습니다.

```text
지하철에서 문제를 푼다
→ 틀린 문제를 표시한다
→ 집 책상에서 오답을 정리한다
→ 카페·도서관에서 긴 복습이나 모의고사를 한다
```

이 루프의 핵심은 다음입니다.

```text
문제풀이
→ 오답
→ 개념 보완
→ 다시 문제풀이
```

## 9. 이직 준비 루프

이직 준비는 전략과 문서 산출물을 `career-playbook`에서 관리합니다.

이 문서에서는 생활 루틴 안에서 이직 준비를 어디에 배치할지만 정의합니다.

```text
지하철 = 채용공고 탐색 / 저장
집 책상 = 이력서 한 문단 수정 / 경력기술서 보완
카페·도서관 = 지원서 작성 / 면접 준비
```

흐름은 다음과 같습니다.

```text
지하철에서 공고를 본다
→ 괜찮은 공고를 저장한다
→ 집 책상에서 이력서/경력기술서를 고친다
→ 카페·도서관에서 지원서를 제출한다
→ 면접 준비 자료를 만든다
```

이직 준비 루프는 다음입니다.

```text
공고 탐색
→ 경력 정리
→ 문서 수정
→ 지원
→ 면접 준비
```

회사 자리나 회사 PC에서는 이직 준비를 하지 않습니다.

---

## 10. 자격증과 이직 준비의 반복 순서

요일을 기준으로 하지 않고, 실행 가능한 장소에 도착했을 때 다음 순서를 실행합니다.

기본 루프:

```text
자격증
→ 자격증
→ 이직 준비
→ 회복
→ 반복
```

집 책상이나 카페에 앉을 기회가 생겼을 때 다음 순서를 하나 실행합니다.

```text
이번 순서가 자격증이면 자격증 25분
이번 순서가 이직 준비이면 이직 준비 25분
이번 순서가 회복이면 필수 루틴만 하고 쉼
```

## 11. 작업 큐

### 자격증 큐

```md
- [ ] PMI Practice Question 10개
- [ ] Mini Test 1개
- [ ] 틀린 문제 3개 오답정리
- [ ] 약한 영역 개념 1개 정리
- [ ] 모의고사 복습
```

### 이직 준비 큐

```md
- [ ] 채용공고 3개 저장
- [ ] 이력서 한 문단 수정
- [ ] 경력기술서 프로젝트 하나 정리
- [ ] 지원할 회사 리스트 업데이트
- [ ] 면접 예상 질문 3개 답변 작성
```

장소에 도착하면 다음처럼 선택합니다.

```text
지하철이면 → 가벼운 문제풀이 또는 공고 탐색
집 책상이면 → 자격증 오답정리 또는 이직 문서 작업
카페·도서관이면 → 긴 학습 또는 지원서 작성
```

---

## 12. 운영 원칙

1. 시간보다 장소를 우선한다.
2. 장소마다 하나의 기본 행동을 둔다.
3. 자격증과 이직 준비를 같은 날 모두 하려 하지 않는다.
4. 침대는 수면 장소로 유지한다.
5. 본사 점심은 헬스장 루틴으로 고정한다.
6. 다른 근무지에서는 회복 루틴을 사용한다.
7. 지하철에서는 깊은 공부보다 문제풀이와 탐색을 한다.
8. 집 책상은 오답정리와 문서 작업 장소로 쓴다.
9. 실행 기준은 낮게 잡고, 반복을 우선한다.
10. 루틴은 완벽하게 수행하는 것이 아니라 끊기지 않게 유지한다.
