# Life Playbook

생활 루틴, 장소 기반 행동 패턴, 자격증 학습, 이직 준비, 건강 관리, 중장기 목표를 지속 가능하게 운영하기 위한 개인 생활 운영 저장소입니다.

이 저장소의 목적은 더 많은 일을 억지로 추가하는 것이 아닙니다. 반복 가능한 구조를 만들어 생활이 무너지지 않게 하고, 자격증·이직·운동 같은 중장기 과제를 일상 안에 배치하는 것입니다.

---

## 1. 루틴의 핵심

시간표보다 **장소**를 기준으로 행동을 정합니다.

```text
장소가 정해진다
→ 할 수 있는 행동이 정해진다
→ 그 행동을 반복한다
→ 반복이 루틴이 된다
```

요일이나 시간대에 지나치게 의존하지 않습니다. 같은 시간이라도 장소가 다르면 할 수 있는 일이 다르기 때문입니다.

핵심 기준은 다음과 같습니다.

```text
집 = 생활 정리 / 회복 / 깊은 작업
지하철 = 가벼운 문제풀이 / 공고 탐색
본사 점심 = 헬스장
다른 근무지 점심 = 식사 / 산책 / 회복
카페·도서관 = 긴 학습 / 이직 문서 작업
침대 = 수면
```

생활 루틴의 우선순위는 다음 순서로 둡니다.

```text
1. 반드시 해야 하는 생활 루틴
2. 회복
3. 자격증·이직 같은 프로젝트 루틴
4. 여가
```

## 2. 최종 구조

### 아침

```text
집 욕실/현관 루틴
→ 씻기
→ 옷 입기
→ 필요한 것 챙기기
→ 출근
```

아침의 목적은 성장이나 생산성이 아니라 **출근 실패를 막는 것**입니다.

### 이동

```text
지하철 루틴
→ PMI Study Hall 열기
→ Practice Question 또는 Mini Test 풀기
→ 틀린 문제 표시
```

지하철은 깊은 공부 장소가 아니라 **문제풀이와 가벼운 탐색 장소**로 사용합니다.

### 근무 중

```text
본사 점심이면
→ 헬스장
→ 운동량은 컨디션에 따라 조절
```

```text
다른 근무지 점심이면
→ 식사
→ 산책
→ 회복
```

본사와 다른 근무지를 같은 기준으로 보지 않습니다. 본사는 운동 장소, 다른 근무지는 회복 장소로 둡니다.

### 퇴근 후

```text
집 현관/욕실/주방 루틴
→ 손 씻기
→ 옷 갈아입기
→ 식사
→ 씻기
```

퇴근 후에는 바로 생산적이려고 하지 않고, 먼저 생활을 정상 상태로 돌립니다.

### 밤

```text
집 책상 루틴
→ 자격증 오답정리
또는
→ 이직 준비 25분
```

집 책상은 깊은 처리 장소입니다. 지하철에서 만든 문제풀이 결과와 공고 탐색 결과를 집 책상에서 처리합니다.

### 주말/여유 시간

```text
카페·도서관 루틴
→ 긴 학습
또는
→ 이직 문서 작업
```

카페와 도서관은 긴 호흡의 작업을 처리하는 장소로 둡니다.

## 3. 사용 방법

이 저장소는 매일 자세히 읽는 문서가 아니라, 루틴이 흔들릴 때 기준을 다시 확인하기 위한 문서입니다.

기본 사용 방식은 다음과 같습니다.

```text
매일 = 장소별 기본 행동만 실행
주 1회 = weekly-review 템플릿으로 점검
월 1회 = 생활/건강/자격증/이직 흐름 확인
3개월 1회 = 중장기 목표 조정
```

사용 순서는 다음과 같습니다.

1. 평소에는 README의 **최종 구조**와 **장소 기반 운영표**만 확인합니다.
2. 루틴이 흔들리면 `docs/location-based-routine.md`를 확인합니다.
3. 목표 방향이 흐려지면 `docs/long-term-goals.md`를 확인합니다.
4. 주간 점검은 `templates/weekly-review.md`를 복사해서 사용합니다.
5. 하루 단위 점검이 필요할 때는 `templates/routine-check.md`를 사용합니다.

## 4. 장소 기반 운영표

| 장소 | 기본 행동 | 부가 행동 | 하지 않을 것 |
|---|---|---|---|
| 집 현관 | 손 씻기, 옷 갈아입기 | 가방 정리 | 바로 침대 눕기 |
| 집 주방 | 식사 | 설거지 조금 | 폰 보며 식사 길게 끌기 |
| 집 욕실 | 씻기, 양치 | 빨래 넣기 | 씻기 미루기 |
| 집 책상 | 자격증 오답정리, 이직 준비 | 25분 집중 작업 | 유튜브, 쇼츠 |
| 침대 | 수면 | 짧은 독서 | 공부, 이직 준비 |
| 지하철 | PMI Study Hall 문제풀이 | 오답 표시, 공고 저장 | 깊은 공부, 이력서 작성 |
| 본사 점심 | 헬스장 | 샤워, 식사 조정 | 점심시간을 폰으로 소진 |
| 다른 근무지 점심 | 식사, 산책, 회복 | 짧은 복습 | 헬스장 압박 |
| 카페·도서관 | 긴 학습, 이직 문서 작업 | 모의고사, 면접 준비 | 의미 없는 인터넷 탐색 |

## 5. 자격증 학습 운영

자격증 학습은 장소별로 역할을 나눕니다.

```text
지하철 = 문제풀이
집 책상 = 오답정리 / 개념 보완
카페·도서관 = 긴 공부 / 모의고사
```

PMP 학습의 세부 전략, Study Hall 기록, 오답노트, 시험 준비는 `pmp-study` 저장소에서 관리합니다.

이 저장소에서는 자격증 학습을 생활 루틴 안에 어떻게 배치할지만 관리합니다.

---

## 6. 이직 준비 운영

이직 준비의 전략, 채용공고 분석, 이력서, 경력기술서, 면접 준비 산출물은 `career-playbook` 저장소에서 관리합니다.

이 저장소에서는 이직 준비를 생활 안에서 언제, 어디서, 어떤 단위로 수행할지만 관리합니다.

```text
지하철 = 채용공고 탐색 / 저장
집 책상 = 이력서 한 문단 수정 / 경력기술서 보완
카페·도서관 = 지원서 작성 / 면접 준비
```

---

## 7. 중장기 목표 운영 요약

중장기 목표는 생활 루틴 위에 올립니다.

```text
생활 루틴
→ 건강/회복
→ 자격증
→ 이직 준비
→ 장기 커리어 확장
```

목표는 3개월 단위로 점검합니다.

```text
1개월 = 루틴 적응
2개월 = 실행량 확보
3개월 = 결과 점검과 방향 수정
```

목표가 충돌하면 다음 순서로 판단합니다.

```text
1. 출근과 가정 생활을 무너뜨리지 않는가?
2. 건강과 수면을 심하게 깎지 않는가?
3. 현재 가장 가까운 마감이 무엇인가?
4. 이직에 직접 연결되는가?
5. 장기적으로 경력 무기가 되는가?
```

## 9. 관련 저장소와 경계

| 저장소 | 담당 범위 |
|---|---|
| `life-playbook` | 생활 루틴, 장소 기반 행동, 중장기 목표, 개인 운영 원칙 |
| `career-playbook` | 이직 방향, 공고 분석, 이력서, 경력기술서, 면접 준비 |
| `pmp-study` | PMP 시험 준비, Study Hall, 오답노트, 시험 전략 |
| `work-log` | 실제 업무 기록, 경력기술 후보 기록 |

경계는 다음과 같이 둡니다.

```text
life-playbook = 운영 방식
career-playbook = 이직 전략
pmp-study = PMP 세부 학습
work-log = 실제 업무 기록
```

## 10. 운영 원칙

1. 시간보다 장소를 우선한다.
2. 장소마다 하나의 기본 행동을 둔다.
3. 생활 필수 루틴을 먼저 처리한다.
4. 자격증과 이직 준비를 같은 날 모두 하려 하지 않는다.
5. 침대는 수면 장소로 유지한다.
6. 본사 점심은 헬스장 루틴으로 고정한다.
7. 지하철에서는 깊은 공부보다 문제풀이와 탐색을 한다.
8. 집 책상은 오답정리와 문서 작업 장소로 쓴다.
9. 루틴은 완벽하게 수행하는 것이 아니라 끊기지 않게 유지한다.
10. 중장기 목표는 3개월 단위로 점검한다.
